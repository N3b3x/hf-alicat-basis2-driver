Detailed Description

Public type definitions for the Alicat BASIS-2 driver.

This file is the canonical, hardware-agnostic representation of everything the BASIS-2 instrument can publish or accept:

  • Driver result domain (DriverError, DriverResult).
  • Calibrated gas table (datasheet "Active Gas").
  • Engineering-unit table (datasheet "Engineering Units").
  • Baud-rate ladder.
  • Setpoint-source taxonomy.
  • Decoded data frames (InstantaneousData, MeasurementData, InstrumentStatus).

Everything is expressed in plain types (no exceptions, no STL allocators) so the driver compiles cleanly on bare-metal MCUs.
