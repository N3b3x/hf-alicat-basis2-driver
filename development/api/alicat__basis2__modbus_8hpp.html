Detailed Description

Modbus-RTU framing and CRC-16 helpers used by the BASIS-2 driver.

Implements only the three function codes BASIS-2 supports: 03 — Read Holding Registers 06 — Write Single Register 16 — Write Multiple Registers plus the canonical Modbus CRC-16 (polynomial 0xA001, init 0xFFFF, LSB first) defined by the Modbus-RTU specification.

All routines are pure / constexpr-friendly, allocation-free, and operate on plain uint8_t* byte buffers so the implementation can be reused on any MCU without the STL.
