Detailed Description

template<typename Derived>
class alicat_basis2::UartInterface< Derived >

CRTP base for the BASIS-2 serial transport.

Template Parameters
DerivedThe concrete adapter (e.g. AlicatUartEspAdapter).

The driver dispatches every byte through static_cast<Derived*>(this), so there is zero virtual-call overhead. The compiler can fully inline the adapter calls into the driver's read/write loops.
